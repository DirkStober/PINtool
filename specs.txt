Specs for Software pintool:

-> Inputs:
	-> TLB assoz
	-> TLB size
	-> Page size
	-> num threads
	-> binary
	-> num mem modules 
	-> cores/node
	-> initial data distribution

-> Outputs:
	-> TLB hits
	-> TLB misses
	-> Shared data between cores
	-> movement of pages from node to node
		-> i.e. node 1 accesses a page that is on node 2
